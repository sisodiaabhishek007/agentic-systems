# HR Policy RAG Assistant

This project implements a Retrieval-Augmented Generation (RAG) based HR Policy Assistant using Python, ChromaDB, and OpenAI APIs.

The assistant retrieves relevant HR policy documents and generates grounded responses for employee queries.

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Set your OpenAI API key:
   export OPENAI_API_KEY="your_api_key"

3. Run the project:
   python hr_policy_rag.py
